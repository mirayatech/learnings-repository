## Two main types of numbers:

### Integer

- 4
- 57
- -10

### Floating Point

- 6.1
- 1.333
- 0,0

## Operators

| Symbol | Name           |
| ------ | -------------- |
| +      | Addition       |
| -      | Subtraction    |
| \*     | Multiplication |
| /      | Division       |
| \*\*   | Exponentiation |
| %      | Modulus        |
| //     | Floor Division |

## Equality Operators

- `==` Equal
- `!=` Not Equal
- `>` Greater Than
- `<` Less Than
- `>=` Greater Than or Equal To
- `<=` Less Than or Equal To

### Notice

- one equal sign is when you assing to an variable and double equal sign is when you compare two values

### Examples

- 2 \*\* 3 = 8 (2 to the power of 3)
- 4 \*\* 4 = 256 (4 to the power of 4)

- 10 / 3 = 3.3333333333333335
- 10 // 3 = 3

## Comments

- Single line comment: `#`
- Multi line comment: `""" """`
